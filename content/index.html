Getting Started

Welcome to the template project! This is a starting point for your new project.

To start customizing your site, you can add a /content/index.html file to your project and start editing.

The template loads as a Handlebars Logic-less template, thus supporting most Mustache syntax.

The framework used is @caweb/framework built with Bootstrap, which provides a responsive layout and a wide range of components to help you build your project quickly and efficiently.

How to's

  • Add more pages, you can create additional HTML files in the /content/pages/ directory. Each file will be rendered as a separate page in your project.
  • Use media on your pages, you can place your media files in the /content/media/ directory. You can then reference these files in your HTML pages.
  • Change the favicon, you can place your favicon file in the /content/media/ directory. File name must be favicon.ico.
  • Change the logo, you can place your logo file in the /content/media/ directory. File name must be logo.png.
  • For more information on how to, visit our How To's page.

Guidelines

Follow these guidelines to ensure your project is set up correctly:

  • When referencing local files, use absolute paths and not relative paths.
  • Content should be structured as followed:

Content Structuring

Regular Sections

<div class="container">
  <div class="row">
    <div class="col-#">
      <p>Content goes here</p>
    </div>
  </div>
</div>

Regular Sections w/ Background Color

<div class="container-fluid" style="background-color: blue;">
  <div class="container">
    <div class="row">
      <div class="col-#">
        <p>Content goes here</p>
      </div>
    </div>
  </div>
</div>

Fullwidth Sections

<div class="container-fluid">
  <p>Content goes here</p>
</div>

Fullwidth Sections w/ Background Color

<div class="container-fluid" style="background-color: blue;">
  <p>Content goes here</p>
</div>
